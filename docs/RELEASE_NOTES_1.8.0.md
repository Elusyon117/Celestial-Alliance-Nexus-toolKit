# Celestial Nexus Toolkit 1.8.0 release notes

Version 1.8.0 is a stability, workflow, and performance release. It intentionally avoids redesigning working modules. The principal changes are repository cleanup, reliable data-import validation, current GitHub Actions, resilient service-worker caching, duplicate-ID repair, and low-risk client-side performance hardening.

Contract Finder remains SCMDB-first. The clean package ships a transparent bootstrap state and must be hydrated with the documented release-asset workflow; it never invents mission records. Language Lab behaves similarly until the MrKraken mirror workflow runs.
