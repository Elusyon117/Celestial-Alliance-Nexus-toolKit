# Data, Privacy, and Browser Storage

Celestial Nexus is a static client-side application. It does not include a project-owned backend or database.

## Browser-local information

Depending on the module and browser, locally stored information may include:

- Event Planner operations, units, crew roles, and assignments.
- Item Finder selections and filters.
- Blueprint Finder filters, selected recipe, and quality-lab values.
- Vehicle configurations and engineering preferences.
- Language Pack Lab configuration.
- Wikelo material locker, project plan, completions, and reputation values.
- Mining targets, favorites, and preparation checks.
- Picture Studio project state.
- Cached roster, source responses, and UI preferences.

## Migration warning

A new GitHub repository creates a new Pages URL. Browser storage from the old repository does not automatically appear on the new site. Export critical plans before changing links.

## External requests

Live-data modules contact public third-party services. Those operators may receive ordinary web-request data such as IP address, user agent, referrer behavior, and requested resource. Their privacy and retention practices are outside this repository's control.

## Uploaded images

Images selected in the Picture Studio are processed in the browser by the toolkit. This repository does not include code that uploads those images to a project-owned server. Browser extensions, external image URLs, and third-party services remain outside that guarantee.

## Public repository warning

Do not commit private rosters, API tokens, credentials, operational secrets, or personal information. GitHub Pages content is publicly accessible when published from a public repository.
